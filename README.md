# No-Code ML Model Training Application

## Overview
The No-Code ML Model Training Application is a web-based platform that allows users to train machine learning models without writing code. It simplifies the machine learning workflow by enabling users to upload datasets, preprocess data, train models, and evaluate performance through an easy-to-use interface.

This project is designed for students, beginners, and non-programmers who want to explore machine learning without deep programming knowledge.

## Features
- Upload datasets easily
- Train machine learning models without coding
- Real-time model evaluation and performance metrics
- User-friendly interface
- Simplified machine learning workflow

## Technologies Used
- Python
- Flask
- TensorFlow
- Scikit-learn
- HTML, CSS, JavaScript

## Project Workflow
1. Upload dataset
2. Data preprocessing
3. Model selection
4. Model training
5. Performance evaluation
